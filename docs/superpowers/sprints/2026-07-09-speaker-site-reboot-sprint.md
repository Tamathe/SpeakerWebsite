## Phase 1 - Discovery
**Status:** done
**Context:** The existing Vite and React site is a compact one-page professional profile with verified healthcare AI projects, talks, and biography content. The user wants a complete visual and experiential restart using Sol and Sites, with future photos and videos as first-class material.
**Constraints:** Preserve factual source material and external links, but do not preserve the existing layout or design system. The result should remain tasteful for university, state, healthcare, and speaking audiences while demonstrating advanced interactive web capabilities.
**Decisions:** Treat the current implementation as replaceable. Use the existing content only as an evidence library. Design for progressive enhancement, accessibility, responsive layouts, reduced motion, and easy media replacement. First impression: visionary physician building the future of healthcare, supported by the ability to make difficult ideas understandable. Use a cinematic but legible page with one signature interaction: an explorable healthcare AI field map connecting projects, talks, collaborators, and outcomes. Launch with curated file handoff while preserving a clean path to a private media studio later. Make short video clips the primary visual material: speeches, panels, students, and projects. Muted clips act as living photographs; selected videos open into a focused player with context, transcripts, and related work.
